# Tag Inherit Plugin for Super Productivity
A plugin for [Super Productivity](https://super-productivity.com) that automatically copies tags to subtasks.

When a subtask is created, it will inherit tags from:
1. the parent task, or
2. the first existing subtask (if present).

## Installation
1. Download and compress the files of this repository into a <samp>ZIP</samp> file
2. Open Super Productivity
3. Go to Settings â†’ Plugins
4. Click "Choose Plugin File"
5. Select the created <samp>ZIP</samp> file
6. The plugin will be activated automatically

## Notes
- The plugin only modifies tags when a subtask is created.
- It does not retroactively update existing subtasks.
